Introduction

Classes and objects are the basic building blocks of object-oriented programming.

Earlier structures were used to define data types in the C language.
For example, to represent a complex number, structure of real number and the imaginary number is made. Functions for operations on complex number are written separately. 

So to make things easier and keep related code in some wrapper. So Object-oriented programming came into the picture in which class is the same as structure but with more advanced features.

Class

Class is similar to structure in C. It is a user-defined data type.

Functions that are related to this data type are wrapped in class and called methods that can access and modify variables without passing as parameters. 

A class is nothing but a user-defined blueprint from which objects are created.

Object

The object is an instance of a class. Think class similar to data type and object similar to a variable of that data type.

Objects have attributes and properties in the class defined. Many objects can be created by the same class.

When an object of a class is created, the class is said to be instantiated.

Data Members

Data Members are members that are declared with any of the fundamental types, eg. int, char, float, double, etc., as well as other types, including pointer, reference, array types, bit fields, and user-defined types.

Basically, variables defined in class are called data members.

Access Specifiers

Access Specifiers define how data members or member functions should be accessed from the main function or within the member function. There are 3 types of access specifiers.

1. private: Private members can only be accessed from within the class and friend classes and friend functions. 

2. protected: Protected members can be accessed within the class and from the derived class.

3. public: Public members are accessible from any part of the program.

Member Functions

Member functions are functions that are defined within the scope of the class. Operators are also member functions.

The data members and member functions of the class can be accessed using the dot('.') operator with the object.

Implementation of Class and Object

C++ implementation


#include <iostream>
using namespace std;

// class complex
class Complex
{
    // access specifier
    public:

    // data members
    int real,imag;

    // member functions
    // constructor
    Complex() {}
    Complex(int r, int i)
    {
        real = r;
        imag = i;
    }

    void display()
    {
        if(imag<0)
        {
            cout<<real<<" -i"<<abs(imag)<<endl;
        }
        else
        {
            cout<<real<<" +i"<<imag<<endl;
        }
    }

};

int main()
{
    // declaration of object
    Complex obj1(3,5);

    // accessing member function
    obj1.display();

    // accessing data member
    obj1.real = 4;

    // accessing member function after modifying data member
    obj1.display();
    
    return 0;
}

Python implementation


# class Complex
class Complex:
    # constructor
    def _init_(self, r=0, i=0):
        self.real = r
        self.imag = i

    # member function - display
    def display(self):
        if self.imag<0:
            print(f'{self.real} - i{abs(self.imag)}')
        else:
            print(f'{self.real} + i{self.imag}')


    # member function - increment
    def increment(self):
        self.real+=1
        self.imag+=1

# main 

# Complex object
obj1 = Complex(2, 3)

# Call display() method
# Output: 2+i3
obj1.display()

obj1.increment()

# Output: 3+i4
obj1.display()
