Introduction

Constructors are the special methods that are called or invoked automatically when the object of the class is created.

Key Points:

Name of the constructor should be same as name of the class.
Constructors are always Public.
Constructors does not have any return type.

Types Of Constructors



1. Default Constructors :

Basically Default constructors does not have any parameters


Let see this with an Example.



Example

1.Here we are declared one class named as Student.

2.In that class we created a default constructor, same name as class i.e Student()

3. In Main function first we created the object s1 of class Student and as soon as we create the object constructor will be called and we will get a respected output.

2. Parameterized Constructors :

> In Parameterized constructor we pass an arguments to it.

Let see the example of Parameterized Constructor.

Example

1.Here we are declared one class named as Student.and here we are taken two variables a and b which are public.
2.In that class we created one parameterized constructor, same name as class i.e Student(). and it will take to arguments which are int type.
3.In main function we created object of class Student and pass two integers to it as its a parameterized constructors.and below we simply printed the values.

Implementation

C++ implementation

#include <iostream>
    using namespace std;
    
    class Student
    {
        public:
            int a,b;
    
        Student(int x,int y)
        {
            a=x;
            b=y;
        }
    
       
    };
    
    int main()
    {
        Student s1(5,10);
        cout<<s1.a<<" "<<s1.b<<endl;
        return 0;
    }


3. Copy Constructors:

> Copy constructor is also a member function of class basically it initialize the object of class with the help of another object of same class.
Let see the example of Copy Constructor.

Example

1.Here we are declared one class named as Student. and declared one variable named as Marks.
2.In that class we created a Parameterized constructor, same name as class i.e Student(). and it will take one parameter of type integer.
3.Below that we create a one copy constructor. 4.In main function we created one object s1 of class Student and passed 100 value as paremeter after that we called the copy constructor and pass old object i.e s1 to that copy constructor.

Implementation

C++ implementation

#include <iostream>
    using namespace std;
    
    class Student // class students
    {
    
    public:
        int marks;
    
        Student(int m1)            // Parameterized Constructor
        {
            marks = m1;
        }
    
        Student(Student &m2)      //  Copy Constructor
        {
            marks = m2.marks;
        }
    };
    
    int main()
    {
        Student s1(100);        //    Calling Parameterized Constructor
        Student s2(s1);         //    Calling Copy Constructor
    
        cout << s1.marks << endl;
        cout << s2.marks << endl;
        return 0;
    }


4.Destructors :

> Destructor is also a member function which destructs the objects of the class.
Syntax of Destructor: ~Constructor Name()

Key Points:

Destructors have same name as class but we have to place tild sign (~) before it. Destructors are automatically called when the objects is destroyed.
Destructors does not have any return type.
Destructors does not take any arguments.
Destructors can not be declared statics or const.
Basically
Let see the example of Copy Constructor.

Example

1.Here we are declared one class named as Student. and declared one variable named as Marks.
2.In that class we created a Parameterized constructor, same name as class i.e Student().
3.Below that we create a one Destructor. 4.In main function we created one object s1 of class Student as soon as we created the object, constructor is automatically invoked and after that below Destructor will be invoked.
5. We are kept printing statements in cnstructors and destructors to understand the execution of given constructors and destructors.

Implementation

C++ implementation


    #include <iostream>
    using namespace std;
    
    class Student                                       // class students
    {
    
    public:
        int marks;
    
        Student()                                            // Constructor Created 
        {
            cout<<"Constructor Called Here"<<endl;
        }
        
        ~Student()                                          //  Destructor Created
        {
            cout<<"Destructor Called Here"<<endl;          
        }
    
        
    };
    
    int main()
    {
        Student s1;     // creating object
        return 0;
    }
