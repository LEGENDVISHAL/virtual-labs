Introduction

The word Polymorphism is consist of Poly + morphs which means Many + Forms. A real-life example of polymorphism, a student at the same time can have different characteristics, like a Boy at the same time is a student, a son,or a worker. So we can say that the same person may have different behavior in different situations. This terminology is called polymorphism. Polymorphism is known as one of the important features of Object Oriented Programming

Types of Polymorphism:



Compile Time Polymorphism:

Compile time Polymorphism is achieved by Function Overloading and Operator Overloading.

Function Overloading:

When there are multiple functions with same name but different arguments then these functions are said to be overloaded. Functions are also br overloaded by change in number of arguments or by change in the type of the arguments.

Let see this with an Example.

Example

1. Here we are declared one class named as Student.

2. In that class we created single function named Display

3. This Display function acts differently in three different situation which is property of Polymorphism

Implementation

#include <bits/stdc++.h>
        using namespace std;
        
        // Class Student
        class Student  
        {
            public:
            
            // function with 1 int parameter
            void Display(int marks)
            {
                cout << "Marks= " << marks<< endl;
            }
            
            // function with same name having 1 double parameter
            void Display(double percentage)
            {
                cout << "Percentage= " << percentage << endl;
            }
            
            // function with same name and string parameters
            void Display(string Div)
            {
                cout << "Name of Div is: " <<Div<< endl;
            }
        };
        
        // Main Function
        int main() {
            
            Student s1;
            
            // Which function is called will depend on the parameters passed
            // The first 'Display' is called
            s1.Display(10);
            
            // The second 'Display' is called
            s1.Display(90.50);
            
            // The third 'func' is called
            s1.Display("C");
            return 0;
        }


Operator Overloading:

Operator overloading is also a compile-time polymorphism in which we overload an operator. Basically it is used to perform the operation on theuser-defined data types. For example in C++ we can add the variables of the user-defined data type that is applied to the built-in data types.

Let see this with an Example.

Example

1. Here we are declared one class named as Student.

2. In that class we created void operator ++() function.

Implementation

#include <iostream>
    using namespace std;
    class Student   // creating class Student
    {
    private:
        int marks;
    
    public:
        Student() : marks(10) {}
        void operator++()                        // Defining Function 
        {
            marks = marks + 2;
        }
        void Print()
        {
            cout << "The Count is: " << marks;
        }
    };
    int main()
    {
        Student s1;
        ++s1; // calling of a function "void operator ++()"
        s1.Print();
        return 0;
    }


Run Time Polymorphism:

Run Time Polymorphism is achieved by Function Overriding.

If we define the same function in the derived class which is already available in the Base class then it is known as Function Overriding. In C++. Basically it is used to achieve runtime polymorphism. With help of this you can provide specific implementation of the function which is already provided by its base class

Let see this with an Example.

Example

1. Here we are declared one class named as Car.

2. In that class we overriding Drive() function.

Implementation


    #include <iostream>
    using namespace std;
    // Created base class Car
    class Car
    {
    public:
        // Difining  Drive function in base class.
        void Drive()
        {
            cout << "Driving...";
        }
    };
    class Audi : public Car
    {
    public:
        void Drive() // defined  same function which is already prseny in base class Car
        {
            cout << "Driving Audi...";
        }
    };
    int main(void)
    {
        Audi a1 = Audi();
        a1.Drive();
        return 0;
    }
